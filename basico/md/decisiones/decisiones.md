# **Decisiones**

## **if**

```py
user_name = "Parsec"

if user_name == "Parsec":
    print("True")
```

---

## **else**

```py
user_name = "Parsec"

if user_name == "Pars":
    print("True")
else:
    print("False")
```

---

## **elif**

```py
user_name = "Parsec"

if user_name == "Pars":
    print("False")
elif user_name == "Parsec":
    print("True")
else:
    print("False")
```
