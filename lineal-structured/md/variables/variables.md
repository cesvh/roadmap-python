# **Variables**

### **Variable de tipo String:**
```py
user_name = "parsec"
print(user_name)
print(type(user_name))
```
### **Resultado:**
```
parsec
<class 'str'>
```

### **Variable de tipo Entero:**
```py
user_date = 91
print(user_date)
print(type(user_date))
```
### **Resultado:**
```
91
<class 'int'>
```

### **Variable de tipo Float:**
```py
user_money = 999999999999999.9999999999999
print(user_money)
print(type(user_money))
```
### **Resultado:**
```
1000000000000000.0
<class 'float'>
```

### **Variable de tipo Boleano:**
```py
user_flag = True
print(user_flag)
print(type(user_flag))
```
### **Resultado:**
```
True
<class 'bool'>
```
